So, in practice there is a list of Subjects, each with its own identification code and name. Each subject belongs to a single Department, which in turn has its own code and name to recognize it. The subjects, then, can be part of multiple degree courses, and obviously a degree course is made up of a lot of different subjects. These Degree Courses are described by a code, a name and some other generic information. Each degree course has its own Exam Sessions, which are uniquely identified by the date and course to which they belong, and the time is also known for each one. Multiple graduate programs may refer to a specific Chair, which is something identified by a name and which has information about the office and the major professor. Each chair is linked to various Teaching Assignments. This Teaching Assignment is a kind of link sheet that binds a chair and a teacher together. The teachers, for their part, have their own number, name and surname. Furthermore, each assignment is associated with a specific Academic Role, which is a simple label with a code and a name, like researcher or something like that. Finally, exam sessions take place in one room, and multiple exams can be held in the same room, but each session has its own. The classrooms are simple, they only have a code and a name to understand which ones they are.
