So, basically we have a system that keeps track of a few things, like stores, brands and categories. For shops, there is a unique code so as not to confuse them, and then the usual information: what the shop is called, the address and which floor it is located on. Then there are the brands, they also have their own code to identify them and, obviously, the name of the brand. Finally, there are categories, which are a bit like large groups; they too have an identification code and a name to understand what kind of stuff we are talking about. The interesting thing is how they relate to each other. Each brand can only belong to one category, it cannot fit in multiple places, but a category, on the contrary, can contain a lot of different brands. And then, a single brand can be sold in a bunch of stores, but each store is specialized and carries only one brand of stuff, as if they were single-brand stores or something.
