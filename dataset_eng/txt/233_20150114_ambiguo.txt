So, in practice we have these large warehouses, each with its own code to recognize it and an address to know where it is. Inside each warehouse there are a lot of shelves, and these also have their own unique code so as not to confuse them. Logically, a warehouse can contain an infinite number of shelves, but a shelf always belongs only to a single warehouse. Then there are the people who work there, the operators, and each of them has a serial number, their identification number. Of course, we also have the products, all the merchandise, and each item is identified by a SKU code. Every now and then we take an inventory to count the stuff; these counting events are recorded, probably using the date as a reference, and it is noted how many pieces have been counted. The interesting thing is the link between the shelves and the inventories: it seems that a single inventory can cover a bunch of different shelves, and, at the same time, the same shelf can be checked during many different inventories over time, creating a kind of cross-connection.
