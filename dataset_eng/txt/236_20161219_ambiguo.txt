So, we basically have a system that organizes some college stuff. We start from the big one, with the States, each with its own acronym and name. Inside each state there are a lot of regions, and obviously a region belongs to only one state. These regions have their own identification code and name. Then, scattered throughout these regions, we find the universities. Let's say that a region can host multiple universities, but each university is fixed in only one region. Each university has its own specific code to recognize it and, of course, a name.

Going into more detail, every university is made up of Departments. A department is identified by its name, but that name alone is not enough, you always need to know which university it belongs to to avoid confusion. We also know more or less how many classrooms there are in these departments. In turn, the departments manage the various degree courses. So, a department may have several courses under it, but a specific degree program is offered by only one department. These courses have their own code, a name and the year in which they were created.

Finally, there are the teachers. Each teacher has his own serial number, a number to identify him and that's it. The somewhat particular thing is that each teacher seems to be connected to multiple degree courses, as if he were a kind of manager, while each degree course refers to a single teacher.
