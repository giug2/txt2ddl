So basically we have a system for tracking cars. We start from the large `Car Manufacturers`, the giants, each with its own identification code and name. Then, each parent company has a few 'Brands' under it, and each brand is closely linked to its parent company, so much so that both its personal code and that of the parent company are used to identify it. Of course, every brand also has a name. Going further down, for each brand there are various car `Models`. A model belongs to only one brand, and here too, to recognize it, a mix of its ID and that of the brand to which it is connected is used, in addition to its name. Finally, there is the `Engines` part, i.e. the engines and versions. Here things get a bit tangled: a model can have a lot of different engines, and a certain engine could be mounted on multiple models, it's a kind of cross-relationship. Each engine option has its own name and code, but is always linked in some way to a specific model, so much so that it uses the model ID to be uniquely identified.
