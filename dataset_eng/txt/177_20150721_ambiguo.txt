So, in practice we have Events, such as conferences, each with its own identification code. Each event is made up of a lot of Sessions, which would be various talks or meetings. A session is only within one event, it cannot be two at the same time. To understand which session we are talking about, you need to know which room it takes place in, which event it belongs to and which topic it deals with; only by putting together these three pieces of information can you identify it without doubt. Each session, in fact, is connected to only one topic, but the same topic can come up in quite a few different sessions. Then there are the Speakers, the people who speak. Here things are very free: a speaker can speak in many sessions and, vice versa, a single session can have more than one speaker on stage, perhaps for a debate. Finally, there is some Material, such as slides or PDFs. Apparently, a session only brings with it one type of material, but the same material could be reused around multiple different sessions.
