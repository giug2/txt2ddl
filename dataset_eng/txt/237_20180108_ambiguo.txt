So basically we have some stuff that describes how a university works. There is the Faculty, which is like a big container with its own code and a name, and inside you find a lot of Department. Each department, which also has a code and a name, is headed by a single faculty. Then, each department manages some Degree Course; think Engineering, Literature, and so on. These courses are identified by a code, have a name, a type of degree and the year in which they started. The rule is that a department can have multiple courses under it, but each course belongs to only one department. Students are enrolled in these courses, each with their own matriculation number which makes them unique. We know their name, surname, grade point average and the credits they have accumulated. A graduate program may have a lot of students, but each student is tied to only one course at a time. Finally, there is a kind of label for each student, called Student Status, which with a code and a description tells us whether he is in progress, out of progress, graduated, or who knows what else. Each student has only one associated status at a given moment, although obviously the same status, such as "active", can be shared by many students.
