In practice, we have all the organization of a university. We start from the campuses, which are the physical places, each with its own code and name to identify it. The interesting thing is that each campus seems to be headed by a single department, but a department, which in turn has a name and a code, can be spread across multiple campuses. These departments then manage the various degree courses; each course has a name that identifies it, you know whether it is a three-year course or a master's degree, and obviously it is linked to a single department. Inside each degree course there are a lot of subjects, or courses, and each subject has its own specific code, a certain number of CFU credits and belongs only to that degree course. For each subject, exam sessions are then organised, each with its own identifier and the reference academic year. Students must register for these sessions, and each booking is recorded with the date and a direct link to that specific session. Finally there are the students, each recognized by their student number, as well as their name and surname. A student, of course, can register for a lot of different exams, but each individual registration is tied to one and only one student.
