So, basically we have some stuff related to universities. There is a whole geographical hierarchy: we have the regions, identified by an acronym, which contain a lot of cities. Each city, which has its own code and name, obviously belongs to only one region. Within a city there may be multiple universities, but each university, with its identifier, is in one and only one city.

Each university is then divided into various departments; a department is defined by its name but is closely linked to its university, so much so that two departments with the same name cannot exist in the same university. Of these departments we also know more or less how many offices they have.

Moving on to people and courses, we have the professors, each with their student number, name and surname, and the courses, each with a code, a name and the reference year. It works like this: a professor can teach several courses, but each single course is taught by only one professor. Also, courses are linked to departments a bit loosely, in the sense that one department has a lot of courses, but a course can also be offered by multiple departments at the same time. There is also a curious connection: each professor seems to be associated with a region, and a region can have many professors.
