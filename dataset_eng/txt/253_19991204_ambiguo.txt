So basically we have a system for keeping track of some music stuff. There is a list of artists, each with their own code so as not to confuse them, in addition to their name and surname. Then, of course, there are the songs, which we call tunes, and they also have a unique code and title. Each song belongs to a musical genre, such as rock, pop and so on; each genre has its own name and code, and while a genre can contain a lot of songs, one song only fits under one genre, to keep it simple. It gets interesting when we connect things: one artist can be linked to a lot of different songs, and one song can be done by multiple artists, think collaborations or bands. Then there are the record labels, with their name, headquarters and a code to identify them. When a song comes out, a kind of register is created, a publication, which links that song to a specific label in a certain year. This publishing thing is kind of the focus, and a single label can obviously have a lot of these releases under their belt. Finally, there are the albums, also with a name and a code, and each album is somehow connected to a group of these releases, as if it were a container.
