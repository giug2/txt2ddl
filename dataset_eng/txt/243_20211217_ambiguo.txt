So basically we have some college related stuff here. There are Faculties, each with its own unique code and name, such as Engineering or Literature. Each faculty, obviously, has a lot of related degree courses. These courses have their own identification ID that works together with the faculty code, a name for clarity and it also states how much it costs per year. The faculties, however, do not do what they want, they must submit to Ministerial Directives, which are a kind of rules from above; a faculty can be linked to multiple directives and a directive can apply to a lot of different faculties, it's a nice mix.

Returning to degree courses, there are also scholarships for students. Each has its own code and name, and the interesting thing is that a scholarship can be valid for multiple courses and, vice versa, a course can give access to several scholarships. When a grant applies to a certain course, we also keep track of the Contribution, i.e. how much money it is.

Then, within each degree course there are the Teachings, in short, the subjects. Each has its own code and a name to recognize it. These courses are managed by the Departments, which also have a code and a name. The matter here is a bit particular: there is a department that is mainly responsible for a teaching, and we also write down the name of this person. However, often, other departments can also collaborate on a single teaching, so there is another type of link where we note down who the collaborators are, and this works in both directions: a teaching can have multiple departments working on it and a department can collaborate on different subjects.
