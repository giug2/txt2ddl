So, in practice we have 'Houses', which are like car manufacturers, each with its own name that identifies it and an address. Each house, obviously, produces a lot of different 'Models', and to recognize a model we need to know its name and which house it belongs to, as well as the year of production. These models are then grouped into a kind of category, which we call 'Segment', identified by its code and description, and many models can be part of the same segment. Going into more detail, for each single model there are different 'Versions', which are a bit like trim levels: to identify them we use their name together with that of the model to which they are connected. For each version we are interested in knowing the price, the number of ports and the speed. Finally, there is the 'Engine' part, which is a thing in itself with its own code, and of which we know the displacement, the number of cylinders and the power. The interesting thing is that a lot of different versions can end up using the exact same engine.
