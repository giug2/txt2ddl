So basically here we have a system that handles a few things on the machines. There are the "Houses", which would be the producers, like the brands, and each has its own identification code and name. These "Houses" are linked to a "Nation", so each manufacturer comes from a certain country. A country can have a lot of producers, but a producer is always from only one country. Each nation has its own acronym and full name. Then, each "House" churns out quite a few "Models". A model is defined by its name but is also directly linked to the house that makes it, so you can't go wrong. One manufacturer has many models, but one model belongs to only one manufacturer. These models, in turn, are placed in a "Category", such as 'sedan' or 'sports'. A category can contain a lot of different models, but a model only fits in one category. Going into more detail, each "Model" is divided into various "Versions", which are specifications such as the set-up or engine, with data such as the displacement. A model can have a bunch of versions, but one version is always just that one model. Finally, there are the actual "Cars", the ones that are on the road, each with its own unique license plate and year of production. Every single car corresponds to a precise "Version", so there can be many identical cars that are of the same version, but a car is of one version and that's it. And then there are the "People", registered with their tax code, name and surname. The interesting thing is that one person can own several cars, and a single car can be owned by multiple people, like in a family. It's a bit of a cross-connection, where everyone can be connected to everyone.
