So, basically here we have a few things that revolve around the university. There are Departments, which are large areas, each with its own code and name. A lot of Professors, each with their own personal code and name, report to some department. Then there are the Sectors, also identified by a code and a name, which seem like another way to group things. In fact, a sector has quite a few professors who work there and, in turn, brings together different Disciplines. A discipline, such as "General Physics", is identified not only by its code but also by the sector to which it is linked, so it is quite specific. From here we move on to the actual subjects; a subject is a somewhat strange thing, to identify it you need a suffix and a direct link to the reference discipline. It also seems that one subject can be a kind of prerequisite for many others. Finally there are the Courses, those that are attended, each with its code, a piece of data called CS and the semester. Each course has its own reference professor, although a professor can obviously teach a lot of different courses. The interesting thing is that a single subject can be part of the syllabus of quite a few different courses.
