So, basically we have a system that tracks a few art-related things. There are nations, identified by an acronym, and obviously each nation has a lot of cities inside it. Each city has its own name and some type of ID so as not to confuse them, and is necessarily within a single nation. Then, in a city there can be several museums, each with its own identification code and name. Inside each museum we find the various rooms; a room is defined by its name and the museum to which it belongs, so you cannot have two rooms with the same name in the same museum, and of these rooms we also know how big they are. Naturally, there are quite a few works of art displayed in a room, and each work is placed there, in that specific room. For each work we know a code, the title and more or less when it was made. Finally, there is the connection with the artists, which are only stored with a code. Apparently, each work of art is linked to only one artist, but one artist may have created a bunch of different works.
