So, basically here we manage a bit of stuff related to people, which we call Assisted Persons. We have the usual information for each one, such as name and surname, but the important thing is that each one has its own unique code to avoid confusion. Each client can be linked to a lot of Practices, which are a kind of files or folders. Each file also has its own identifying number that is never repeated and a date, more or less than when it was opened. Inside each of these practices, then, there may be different items, which would be the Performances. Think of these services as the individual lines of a document: in fact, to identify them, we use the number of the case to which they belong together with a progressive line number. For each service we write down a description to understand what it is about, the date on which it was performed, an amount if any, and also an indication of a certain Subject involved in that specific activity.
