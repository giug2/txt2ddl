So, we basically handle a little bit of sales related stuff here. There is a list of customers, each with their own identification code to avoid confusion, and then we write down the name and address. Clearly, a customer can place a lot of orders over time, but every single order belongs to only one person. Each order has its own number to be tracked and a total amount to understand how much it is worth. When an order leaves, it is entrusted to a courier; we have a list of couriers, each with its own name and code, and one courier can obviously manage many shipments, but each order is managed by only one courier. Inside an order there is the detail, that is, the individual lines that make it up. An order is made up of multiple lines, and each line has its own identifier, the price of that piece and the date it was added. Finally, each line of detail refers to a specific product in our catalog. Products have a code and a name, and the same product can be sold an infinite number of times in different orders, but each detail line only ever points to a single type of product.
