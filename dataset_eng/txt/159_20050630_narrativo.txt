Imagine a vibrant digital ecosystem, designed for the complete management of online learning offerings. At the center of this universe is the E-learning Platform, the beating heart of the system, identified by a unique code, a name, a web address and a support contact to assist users. This platform is not just a container, but the stage on which the entire learning process takes place. The protagonists of this stage are the Teachers, key figures defined by an identifier, name, surname and date of birth, with the possibility of also recording a collection date to manage their activity cycle. Each teacher, with his experience and knowledge, is the author of one or more courses, creating an indissoluble bond in which a teacher can create an entire library of contents, but every single course has a single reference author. Each Course is in turn a well-defined entity, with its own identification code, a Title that captures its essence and the year of publication that contextualizes its relevance.

However, a course is an abstract concept until it is made available to students. This is where the Course Sessions come into play, which represent the actual editions of a course in a given period. A single Course can be delivered countless times, giving rise to multiple Sessions, each with its own identifier and, above all, a start date and an end date that define its duration. Each Course Session is hosted and managed by a specific E-learning Platform, which can simultaneously host a vast catalog of different sessions.

Access to this educational wealth is regulated by a flexible and structured tariff system. The E-learning Platform defines different subscription options through the Rates. Each Tariff, identified by its code, represents a specific commercial offer and is the result of the combination of two fundamental elements. On the one hand, we have the Access Level, which establishes what content or functionality the user is entitled to, described by a code and a clear description (for example, "Basic" or "Premium"). On the other hand, there is the Subscription Type, which defines the time mode of the subscription, also with a code and a description (such as "Monthly" or "Annual"). A single Tariff is therefore associated with a single level of access and a single type of membership, but the same level or type can be used to create many different tariffs. To make the offer even more dynamic, the system also provides for the management of discounts. The E-learning Platform can offer various Types of Discounts, each with its own code and description, and at the same time, a certain type of discount could be applied on multiple platforms, creating a flexible relationship that allows you to manage targeted and versatile promotional campaigns.
