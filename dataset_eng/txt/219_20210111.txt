Show a conceptual scheme for a database that manages information relating to different universities.
The following aspects are of interest:
- the universities, each with an acronym and full name
- the departments, each of which belongs to one and only one university and has a code (unique within the university), name and location
- degree courses, each of which is offered in zero or more departments, has a unique code and a name and pertains to one and only one faculty
- the faculties, each of which has a code (identifier) and name
- the teachers, each of whom has a serial number (identifier), surname and name and belongs to zero or more departments, at each of which he has a classification (a simple value, which can be different from one department to another); each teacher was born in a municipality and resides in a municipality
- the municipalities, which have an ISTAT code (identifying) and name
