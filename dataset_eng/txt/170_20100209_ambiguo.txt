So basically we're keeping track of some stuff about cars here. There is this main thing, the Vehicle, which is uniquely identified by its License Plate. In addition to that, we note down who the Owner is, where he lives, and a couple of data on the kilometers, such as how many he had at the beginning and how many he has now. Then, each vehicle can be connected to a lot of Interventions, think of the work done in the workshop. Each intervention has its own code to avoid confusion, the date it was done, some scattered notes and obviously the cost. But it doesn't end here, because each intervention is not generic, but belongs to a category. There is a kind of list, the Intervention Type, which has its own code and a description to explain what type of job it is, such as "oil change" or "bodywork repair". In practice, one type of intervention, such as an "oil change", can be associated with many different interventions carried out on different machines on different dates.
