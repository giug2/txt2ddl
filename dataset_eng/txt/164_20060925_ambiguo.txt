So, basically there's this festival stuff, which is the main events. Each one has its own code so you don't get confused, a name, and they tell you when it starts and when it ends. The heart of everything are the performances, which are practically concerts: each performance is linked to a single festival and a single artist, and these two things together make it unique. Obviously there are the artists, who have their own identification code, a musical genre and the year in which they started playing. An artist can do a lot of different performances, but each performance is by only one artist. Then there are the stages, each with its own name, a code and an indication of how many people can fit there. A lot of performances can be held on one stage, but every single performance happens on a specific stage. To organize things a bit, they invented thematic paths, which are kind of strands or themes identified by a code. This becomes interesting because a performance can be part of multiple paths, and a path is made up of many performances lined up; in fact, the order of the stages of each performance within a route is kept track of. These thematic routes are also used for another thing: guided tours. Each route may have several associated guided tours, but each tour follows one and only one route. These tours have a name, a departure time and a duration. Finally, for each type of tour, specific groups are created on certain days, and for each of these groups it is known how many have registered for that date.
