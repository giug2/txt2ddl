So basically we're keeping track of a few things about skiers here. Each skier has his own code to avoid confusion, most obviously the name and surname. Then, each skier is linked to only one nation, his own, and also to a specific city of that nation, probably where he comes from. Countries are simply lists with a code and a name, and each country can have a lot of cities and obviously a lot of skiers. A city, however, is only in one nation. There are also various ski specialties, such as downhill, slalom and so on, each with its own name and code. Things get interesting when we connect skiers to specialties: a skier can be good at multiple disciplines, and clearly many different athletes do one discipline. In this connection, we also note the number of victories, so we know exactly how many times a certain skier has won in a certain specialty.
