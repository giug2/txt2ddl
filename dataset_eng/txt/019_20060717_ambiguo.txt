So, basically, the hub of everything seems to be the College. Each college has its own basic information: a code, a name, where it is located and a phone number. These colleges are spread across various locations, and each location can accommodate quite a few colleges, although obviously one college is fixed in one place. To get to these locations, airports are involved; each place has its own reference airports, and an airport serves that specific area. From these airports you can depart or arrive from a lot of different cities, and vice versa, a departure city is connected to multiple airports, so there is quite a lot of comings and goings. Coming back to colleges, each of them has its own offerings. For example, they offer various types of stay, each with its own period, price and a certain number of hours, and these options are specific to that college. The same goes for exams: each college has its own list of exams, each with a cost and a name, which you can only take there. Finally, there is a kind of discount system, or 'reduction types'. A college can apply several discounts, and a single discount can be valid for multiple colleges, so there is a more flexible connection. In general, almost all these things, whether they are airports, cities, exams or locations, have their own identification code and a name to understand something about it.
