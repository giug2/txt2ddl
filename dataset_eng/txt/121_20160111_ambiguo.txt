So, basically we're talking about cars here. Every single car is unique, identified by its chassis, and then we note down the color and some date. Each car produced belongs to a specific version, you know, like the "luxury" or the "sport" version. We keep track of these versions with a code, and for each one we know the name they give it, the displacement and the list price. Clearly, a version does not exist alone, but is linked to a car model. A model, which has its own code to avoid confusion, has its own characteristics such as length and number of seats. A lot of different versions can be part of the same model. Furthermore, each version has its own fuel type, which is recorded with a code and a short description, such as "petrol" or "electric", and obviously a fuel type can be used for many different versions. Finally, the models themselves are grouped into market segments, also with a code and a description to understand what we are talking about, such as "SUV" or "utility car". A market segment may contain a lot of models, but each model belongs to only one segment.
