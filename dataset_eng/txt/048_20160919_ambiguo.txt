So, basically there's a whole system for keeping track of the cars. We start from the BRAND, which is simply the manufacturer, identified by its code and name. Each brand, as we know, comes out with quite a few different MODELS. Each MODEL also has a code to recognize it, a commercial name, and then there is information such as how long it is and how many seats it has. These models are then put into a sort of category, which we call MARKET SEGMENT, to understand if it is a small car, an SUV or something like that; This category also has its own code and description.

Going more specifically, for each MODEL there are different VERSIONS. The version is like the setup, with its own code, a name that distinguishes it, the engine displacement and the list price. Obviously, a certain version only ever belongs to one model. Furthermore, each version is linked to a FUEL TYPE, which basically says whether it runs on petrol, diesel or other, and this also has its own code and description.

At the end of it all there is the physical CAR, the one you buy. Every single car is an instance of a certain version, so a lot of identical cars can come out of one version. Each car is unique thanks to its CHASSIS number, and then you know what COLOR it is and the DATE of production or something like that.
