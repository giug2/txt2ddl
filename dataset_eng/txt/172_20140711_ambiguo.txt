So, in practice we have places, let's call them warehouses, each with its own code to recognize it and an address to know where it is. A few people work inside each warehouse, the employees, and each of them has a serial number and a name. The important thing is that each employee is fixed in a single warehouse, it's not like they jump from one place to another. These employees are then responsible for managing the batches of goods. Each batch has its own identification code and production date to know when it was made. A single employee may have managed a lot of different batches, but each batch can be traced back to only one person. Finally, there's the actual stuff, the product, which has its own SKU code and name. The same type of product can be present in many different batches, but within a single batch there is always only one type of product, you don't make minestrone.
