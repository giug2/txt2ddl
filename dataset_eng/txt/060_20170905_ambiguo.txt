Basically, we have a system that tracks a few things about orchestras. There is a kind of archive of the various formations that an orchestra has had, and the thing that distinguishes them is the year, so every year there is a different composition. Then, obviously, there is a list of all the musicians, and each one has its own identification code to avoid confusion. There is also a list of roles, you know, like first violin, oboe and stuff like that, each with its own code and name to clarify. The central point is a table that puts everything together, a kind of participation register. In there it says that a certain musician was part of a certain band of a given year. So, a single annual lineup is linked to a lot of holdings, one for each musician, and similarly a musician can have a lot of holdings over the years. In this same participation table, the role that that musician had on that specific occasion is also specified, so a role, such as "cello", can be associated with many different participations.
