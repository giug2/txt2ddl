So, basically here we keep track of some stuff about books and who makes them. There are publishing houses, each with its own name and a code to recognize it. Strangely, each publisher seems to be tied to just one language, even though one language can be used by lots of different publishers. Each publishing house, obviously, has its own book series, and a series belongs only to a specific publisher; in fact, to be sure which series we are talking about, you need to know both its name and the publisher.

The series are also classified by genre, you know, like "mystery" or "science fiction", so each series falls into a single genre, whereas one genre can include a lot of series. Within a series there are the various editions of the books, each with its own identification code and number of pages, and clearly each edition is part of one and only one series. From a single edition, then, all the physical copies derive, that is, the actual books that can be touched, each with its own inventory code so as not to lose it and the year in which it was printed.

Finally, there are the people, the users, registered with name, surname and their personal code. The interesting thing is the connection with books: a person can borrow multiple physical copies and, at the same time, a single copy can change hands between many different users over time.
