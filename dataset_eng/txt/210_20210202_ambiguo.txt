In practice, we have a system that revolves around the Products, each with its own unique code, the SKU, and obviously a name. People can then leave reviews on these products; each review has its own identifying number, the date it was written and a rating. Clearly, a product can have a lot of reviews, but a review is only linked to that specific product. To keep things in order, each product is placed in a Subcategory, which has its own name and perhaps a description to make it clear what it is about. In turn, each Subcategory is part of a larger Category, which just has a code and a name. So, to understand a subcategory well, you need to know both its code and which main category it attaches to. Basically, a large category can have quite a few subcategories underneath it, and within each of these you'll find a bunch of different products.
