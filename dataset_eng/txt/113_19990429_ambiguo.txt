So, in practice there is a whole structure organized around the Departments, each with its own code and name. Inside each Department, obviously, you will find a lot of teachers, who are also identified by a code and their name. These teachers are not only linked to their department, but also belong to different Sectors, which are a kind of thematic areas, each with its own identification code and description. Each Sector, in turn, groups together a number of Disciplines, which are general study subjects, defined by a code and a name such as "Mathematical Analysis". Various more specific subjects can then derive from a Discipline, which in order to be identified require both their own particular suffix and a reference to the Discipline from which they come. The interesting thing is that a Subject can be connected in some way to other Subjects, perhaps as a propaedeutic or something like that. Finally, there are the actual courses, which are held in a certain semester. Each Course is defined by its own code, the CS, and is closely linked to a Subject, which suggests that the same Subject can be offered in different courses. And, of course, each teacher is responsible for teaching quite a few of these courses.
